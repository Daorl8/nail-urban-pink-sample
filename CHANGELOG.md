## 이력 (요약)

2. 더스티 모브(v2) → "아줌마" 피드백

4. **옵션2: 클린 뉴트럴 + 쿨로즈(v4)** → 채택
5. **각진 레이아웃**(네일 아뜰리에 실측 근거: 해당 사이트는 radius 0개·그림자 1개뿐) — radius 전부 0, 그림자·호버 들림 제거, about 사진 1:1
6. 갤러리 3안 비교(전부 정사각 → 비율 혼합 메이슨리 → **모자이크 확정**). 메이슨리는 열 끝이 들쭉날쭉해 폐기, 모자이크는 4×3 12셀 정확히 채움
7. 시그니처 카드 그라데이션 제거 → 흰배경+로즈테두리+배지
8. 웜 블러시 배경 실험 → **롤백**(쿨 뉴트럴 유지)
